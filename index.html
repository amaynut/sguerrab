Hospital Information System

This is my latest project in web development that I did as a school project to obtain my degree as a Software Application Specialist. I did the project alone. The main goal of the project was to develop a Hospital Information System as a web application. The final product allows the interaction between 4 types of users involved in a hospital work place: a patient, a doctor, a medical assistant and a medical officer. It allows the accomplishment of various tasks related to a Hospital Information System, such as: make a new prescription, make an appointment, manage the list of drugs... The main technologies that I used were: Java Servlet, JSP, JavaScript, JQuery, Bootstrap Framework, HTML5, CSS3...

  • Client: School Project (Vanier College)
  • Date: May - August 2014
  • Service: Web Development
