HTML demos

We can do amazing things with HTML5, especially when it's associated with the power of CSS3. A lot of things what were not thought possible with HTML before. For instance we can make an online presentation (a slide-show) without any plugin. This is what we can see in the following link: HTML5 Can website

In this page we are going to show some demos. The page will add some demos as the time goes. We will try to keep it at least monthly demos to showcase the powerfulness of HTML5 combined with CSS3

Audio and video integration:

HTML5 came with new elements <video> and <audio> to give us the ability to include a video or a sound track with flash or any other plugin.

Insert a video:

The following code


            <video width="320" height="240" controls>
                <source src="video/Nature-Canada.mp4" type="video/mp4">
                Your browser does not support the video tag.
            </video>

Will output the following video (the video can be controlled by the user: pause, play, full-screen, ...):

Note: To make the video works in all modern browsers, including Firefox, the most common format is .mp4

Insert a sound track

The following code will insert a beautiful song from an Algerian singer Matoub Lounes: 

      
        <audio controls>
                <source src="audio/Matoub-Lounes-Zzher-iw.mp3" type="audio/mpeg">
                Your browser does not support the audio element.
            </audio>

Note: For the audio element the most supported format is .mp3

New form elements and attributes:

New attributes

Placeholder attribute for an input field:

A placeholder for an input field must be a text. It disappears when the user enter a new text in the field.

autofocus attribute for an input field:

With autofocus attribute, the field is automatically focused (the cursor goes inside it) when page loads, as we can see in the example above. Browsers that donâ€™t support the autofocus attribute will simply ignore it.

New values for input type attribute:

Email input:

The email type is useful for form control or to display a keyboard with handy @ in case of touch devices with non physical keyboard.

URL input:

Number input

The input of type number is very useful to enter a number and it give as a lot of control, like the minimum value, maximum value, the default value and the step from a value to another, as we can see in the following example.

  <input type="number"
                   min="0"
                   max="10"
                   step="2"
                   value="6">

An number as slider: 

  <input type="range"
                   min="0"
                   max="50"
                   step="1"
                   value="6">

Notice that the slider can also be controlled by the keyboard (left and right arrows)

Date input

The date input will allow us to show the date piker as we can see in the following example: 

  <input type="date"  >

Note: The date input is not supported by all modern browsers. Currently it's only supported by Google Chrome, Opera browser and Safari (for Mac OS and iOS).

A lot of HTML5 elements and attributes are not supported by some browsers, especially INTERNET Explorer which is used to have a poor support for new standards like HTML5 and CSS3. To check which browser supports which element or attribute, please visite the following web site http://html5test.com/
