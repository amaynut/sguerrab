CSS3 Selectors

CSS3 came with a bunch of new and useful selectors to make the selection process easier. Now, with CSS3 selectors we can virually select anything in the HTML document. For instance, we can select elements by specifying the language attribute.

Universal selector: Select all with (*) operator:

table * {padding: 1ex;}

Will select all descendants of the element table. If no parent element specified, the * will cause the selection of everything within the document. It will be the shorthand for html * { ...};

Select an element by its id with (#) operator:

#main  {text-align:  justify;}
The above code could be written by specifying the target element with the given id. For instance, if it's a div, we could write: 
div#main  {text-align:  justify;}

Select an element by its class with (.) operator:

.error  {color:  justify;}

The above code will select any element which has a class named error. An HTML element can belong to several class. If we want to taget only a specific elements with a specific class, for instance, to select a paragraph of class error, we need to write:

p.error  {color:  justify;}

So, the above code won't be applied to div or any other elements which has error as a class name.

Select a descendant element with this syntax: ascendant descendant:

footer p {font-size: 0.9em;}

The above code will selection all paragraphs(p) descendant of the element footer, no matter the deep of this hierarchy. Here the paragraph hasn't to be the direct child of footer element.

Type selector: Select an element by it's name:

article { margin: 1em 4em;}

The above code will select every element named article in the document.

Select an element specifying it's adjacent sibling, with adjacent-sibling + target-element 

div + p  { line-height: 1.5;}

The above will select every paragraph which comes directly after a div element as a sibling. The target element has to come directly after the adjacent-sibling. Like the below example:

         

<div>
I'm a div element, I'm an adjacent sibling of the following paragraph. 
</div>
<p>
Hey, I'm a paragraph. I want to be selected with adjacent sibling notation. 
</p>
<p>
Hey, I'm a paragraph too. but I won't be selected because I don't follow directly the div element. 
</p>

Select an element by specifying its parent with parent > child: 

div >  p  { color: blue;}

The above code will cause the selection of paragraphs which are direct child of a div element.

Select an element by specifying its sibling sibling ~ sibling-target-element: 

span ~  strong  { text-decoration: underline;}

The above code will underline every strong element which is sibling of span element.

Attribute selectors

Select an element by it's attribute with element[attribute]:

td[colspan]  { background-color: #CCC;}

The above code will cause the selection of every td which has a colspan attribute

Select an element by specifying the value of an attribute element[attribute="value"]:

td[colspan="3"]  { background-color: rgb(125, 100, 0);}

The above code will select only the td element with colspan="3"

Select an element by specifying one value of a given attribute element[attribute~="value"]:

This is useful when an attribute has more than one value. For instance, if we have the following code: 

         

<a rel="external copyright">
I'm a an external link to copyright.
</a>
<a rel="copyright">
I'm an internal link to copyright.
</a>

We need to use the following CSS code to select the links with copyright relation.

a[rel~="copyright"]  { background-color: #CCC;}

Select an element by specifying the beginning of its attribute before the dash (-) symbol element[attribute |="value"] :

 a[hreflang|="en"] {color: brown;}

Notice in the above example the presence of | character. Which will cause the selection hreflang="en-US" as well as hreflang="en-CA" for instance.

Select an element by specifying a substring value of an attribute

Begin with : [attribute^=value]

img[src="images/"]  {width: 50%;}

End with : [attribute$=value]

a[href$=".html"]  {color: white;}

Contains at least one instance of the substring : [attribute*=value]

p[title*="hello"]  {width: 50%;}

Pseudo-classes selection

We select an element based on a its pseudo-class, a pseudo-class represent a state of that element (for instance when a link is hovered, the pseudo-class is :hover, it can also represent a part of that element. For instance :first-letter will select the first letter of an element.

The link pseudo-classes: :link and :visited

The unvisited link:

a:link  {color: blue;}

The visited link:

a:visited  {color: black;}

The user action pseudo-classes :hover, :active, and :focus

Those pseudo-class can be applied on any element, not only a link

Select an element when it's hovered: 

p:hover  {color: red;}

Select an element when it's active

The active state is the moment between the click and the release of the mouse button on an element.

button:active  {background-color: yellow;}

Select an element when it's focused 

input[type="text"]:hover  {border-color: green;}

Select an element when it's targeted by an URL with :target :

For the following HTML code: 

         

<div id="section1">
<p>
I'm the first section in the document
</p">
</a>
<div>
I'm another section in the document
</div>

When ever the div element is targeted by fragment identifiers link like example.org#section1, the following code will run:

:target  {background-color: green;}

Select an element by its language with :lang pseudo-selector

The target element must have a language attribute

section:lang(fr) {background-color: green;}

The above code will select any section where the language is set to French

The UI (a form for instance) element states pseudo-classes:

The :enabled and :disabled pseudo-classes

The above pseudo classes are useful for User Interface like a form. The :enabled pseudo-class represents user interface elements that are in an enabled state.

input:disabled {background-color: #333;}

The above code will select the input fields in disabled state and the following code will select the input fields in enabled state

input:disabled {background-color: rgb(0, 200, 0);}

Structural pseudo-classes

:root pseudo-class

The :root pseudo class will select the root element in the XML document. In HTML documents, it's the html tag in most cases.

:nth-child() pseudo-class

This pseudo select is used to select the nth child of the parent element. It will always select the nth child no matter the type of that child. The syntax is parent element:nth-child(n) {style}. n can be a number, a keyword, or a formula With the following example we are going to select the seventh row in a table:

table tr:nth-child(7) {background-color: blue;}

If we want to select the even children we put the keyword even as a parameter. The same thing to select the odd children, we put the keyword odd

Note: Internet Explorer 8 and earlier versions do not support the :nth-child() selector.

:nth-last-child() pseudo-class

This pseudo class select the nth child counting from the last child. For instance to select the last a paragraph which is the before last child of aside element we will need to right: 

aside p:nth-last-child(2) {style}

:nth-of-type() pseudo-class

With the :nth-of-type() pseudo selector we specify the type of the child. For instance if we write: 

aside p::nth-of-type(2) {style}

We will select the second child of aside element of type paragraph only.

:nth-last-of-type() pseudo-class

:nth-last-of-type() is used to select the nth child of a given type from the last child of that type. The following code will select the third child of type paragraph counting from the last paragraph:

div :nth-last-of-type(3)

:first-child pseudo-class

This pseudo code is used to select the first child of a parent element no matter the type of that child.

:last-child pseudo-class

This pseudo code is used to select the last child of a parent element no matter the type of that child.

:first-of-type pseudo-class

This pseudo code is used to select the first child of a parent element of specific type.

:last-of-type pseudo-class

This pseudo code is used to select the last child of a parent element of specific type.

:only-child pseudo-class

Selects every element that is the only child of its parent. The following code will select an element that is the only child of its parent. If the parent element has more that one child it will select nothing.

:only-child

:only-of-type pseudo-class

This is used to select the only child of a given type.

p:only-child {style}

The above code will select the only child of type paragraph.

:empty pseudo-class

This is used to select an element that has no children (including text nodes). For instance to select an empty table cell, we need to right:

td:empty {background-color: red;}
