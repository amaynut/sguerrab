Presentation of HTML5 and CSS3

What is HTML5?

HTML5 is the latest version of HTML Standard. Which is used to structure the presentation of web pages. As of February 2014, is a candidate recommendation of the World Wide Web Consortium (W3C). As its stated in the W3C official website: HTML5 defines an HTML syntax that is compatible with HTML4 and XHTML1 documents published on the Web. Source

The W3C is planning to release the stable recommendation for the version 5.0 by the end of this year (2014). But the work for the future version (5.1 and 5.2) has already begun. Hereafter a table which shows the calendar for the release of the different versions of HTML5:

The combined timelines for HTML 5.0, HTML 5.1 and HTML 5.2 (Source: Wikipedia):

What's new in HTML5?

HTML5 became rapidly very popular because it came a whole new bunch of staff. Hereafter a short list of new things brought by HTML5:

  • New elements (new tags) such as: <video>, <audio>, <nav>,...
  • HTML5 is much more semantic. It uses elements which has a meaning in the document: <article>, <header>, <footer>,...
  • New Attributes: A lot of useful attributes are introduced by HTML5 to make the web development easier.
  • Video and Audio: Now we can add a video or audio without an external plugin like Flash.
  • 2D/3D Graphics: Now we can draw directly in HTML5 webpage with the help of technologies such as: CSS3, SVG and JavaScript
  • Local Storage
  • Local SQL Database
  • Localisation

What's CSS3?

CSS3 is the third and the latest version of CSS which is the language to format the HTML content of a web page. CSS is designed to separate presentation (look and feel) from the actual content (text and medias). That's why former HTML elements like <font> or <center> (present in HTML4) are now depreciated in favor of CSS use.

Historically, the fist version of CSS was released as a recommendation by the W3C in 1993. Unlike the previous version of CSS (1 & 2) which were made as one big monolithic block, CSS3 is completely modular. In other words, CSS3 specification is a set a separate modules. Each module add new features or extend features defined in CSS2.1 with backward compatibility. Due to this modular character, different modules have a different status. What's new in CSS3?

CSS3 brought a lot of new capabilities to style a markup document. Hereafter a shortlist of few of them:

  • New selectors: with CSS3 module the selection capabilities is much more broad. For instance, now we can select an element depending on its attribute.
  • Border styling: with CSS3 we can easily style borders with rounded corners for instance without need of an image.
  • New color capabilities: CSS3 a lot of new features related with usage of colors, such as gradients and opacity
  • Text effects: CSS3 offers a large possibilities to make a sophisticated text-effects with properties like: text-shadow, text-overflow and word-wrap.
  • Multi-column layouts: As we seen in the previous table, CSS3 has a module dedicated to multi-column which offers an easy way to make a multi-columns layout just as press magazines of newspapers are doing since decades.
  • Web fonts: with @font-face statement, CSS3 allow us to import easily beautiful fonts and use them directly in the web site.
