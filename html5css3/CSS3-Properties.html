CSS3 Properties

CSS3 updated, outdated some of CSS2.1 properties and added a lot of new properties to do some cool staff that any web designer would love to do with CSS only like: transition, animation, multi-column and so on. Hereafter a table for most important properties brought by CSS3:

Property Possible values
Rounded corner
The short-hand border-radius:
The individual border setting border-top-right-radius,
border-bottom-right-radius,
border-bottom-left-radius,
border-top-left-radius
Possible values: Any unit of measure (px, pt, em, ex, %, ...)
Box-shadow
For Webkit browsers (Safari, Opera, Chrome...) -webkit-box-shadow
For Mozilla (Firefox) -moz-box-shadow
Standard (W3C) box-shadow
Syntax x-offset, y-offset, blur, color
Example 5px 10px 3px #251542
Box-shadow
Webkit -webkit-column-count
-webkit-column-gap
Firefox -moz-column-count
-moz-column-gap
Standard column-count
column-gap
Text shadow
Standard Property text-shadow
Syntax x-offset, y-offset, blur, color
Example 5px 1px 3px brown
Microsoft version filter:dropshadow()
Syntax color, x-offset, y-offset
Example filter:dropshadow(yellow, offx=5, offy=8)
Transparency
syntax background-color: rgba( Red, Green, Blue, Alpha)
color: rgba( Red, Green, Blue, Alpha)
Alpha defines the opacity by a value between 0 (full transparency) and 1 (full opacity)
Example background-color: rgba(100, 25, 0, 0.3)
Import a font face
Import declaration @font-face
Syntax font-family: "myFontFamily"
src: url("myFontFile.woff") format("woff")

In the following days we are going to add new properties and values to the above table, because the list of new staff in CSS3 is long!
