HTML5 elements and attributes

HTML5 Elements

The new version of HTML came up a lot of new elements to structure the document from the largest elements to create for instance sections in the document to the smallest elements to markup, for instance, a small portion of a text. The HTML5 specification, which is not yes a final recommendation, divides those elements into various categories depending on their role in the document. Hereafter a table which shows a few of them classified in their appropriate categories.

HTML5 Attributes

HTML5 attributes are divided in two categories: global attributes and non global attributes. Global attributes are commons for all HTML elements, in other words, they can be added to any HTML elements. On the other hand, the non-global attributes are specific to one or more elements.
